# Enact

Skills framework for AI agents. Each skill under `skills/` is a self-contained
guide that agents can discover and use via Claude Code or the Anthropic API.

## Usage

Symlink into your Claude Code directories:

```bash
ln -s /path/to/enact/skills ~/.claude/skills
ln -s /path/to/enact/agents ~/.claude/agents
```
